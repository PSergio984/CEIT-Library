# CONVENTIONS.md

## Naming & Style
- **PHP**: PSR-12 (enforced by Laravel Pint).
- **Variables/Methods**: Descriptive camelCase (e.g., `hasAdminAccess`).
- **Models**: Singular TitleCase.
- **Livewire**: Components in `app/Livewire/Pages/`, views in `resources/views/livewire/pages/`.

## Code Standards
- **Strict Typing**: Explicit return types and parameter hints required.
- **Control Structures**: Always use curly braces.
- **Constructors**: Prefer PHP 8 constructor property promotion.
